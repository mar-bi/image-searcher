Image Searcher Microservice

User Stories

  1. I can get the image URLs, alt text, and page URLs for a set of images relating to a given search string.
  2. I can pagginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted strings.

How to use

  • Search:
    http://something-fancy.glitch.me/api/imagesearch/funny%20bears?offset=4
    https://azure-mile.glitch.me/api/imagesearch/crazy mice
  • Retrive the search history:
    https://azure-mile.glitch.me/api/latest/
